# Desafio-DISYS | NodeJs Server
 Desafio de desenvolvimento Back-End.
 CRUD básico para cadastro de alunos.
 
## Comandos para a instalação e start do projeto com (ReactJS).
 
 - npm install
 - yarn
 - yarn build
 - vercel
 - vercel --prod

 ## Para Start no projeto local:
 
 - nodemon server.js

## Versão no Node.

 - versão 20^
